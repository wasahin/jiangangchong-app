# PPTX

## Description

Presentation creation, editing, and analysis. Work with .pptx files for creating new presentations, modifying content, working with layouts, and adding comments or speaker notes.

## When to Use

- Creating new presentations from scratch
- Modifying or editing existing .pptx files
- Working with slide layouts and templates
- Adding comments or speaker notes
- Converting content to presentation format

## Instructions

1. **Structure**: Start with a title slide, then content slides with clear headings
2. **Visual consistency**: Use consistent fonts, colors, and spacing throughout
3. **Content density**: Keep each slide focused on one main idea
4. **Visuals**: Use charts, tables, and images to support key points
5. **Speaker notes**: Include speaking points in the notes section

## Slide Types

- **Title slide**: Presentation name, subtitle, presenter info
- **Content slide**: Heading + body text or bullet points
- **Chart slide**: Data visualization with supporting text
- **Image slide**: Full-bleed image with overlay text
- **Section divider**: Topic transition with section name
- **Summary slide**: Key takeaways or next steps

## Best Practices

- Use no more than 6 bullet points per slide
- Keep text concise — slides support the speaker, not replace them
- Choose readable fonts at appropriate sizes (minimum 18pt for body)
- Use high-contrast color combinations for accessibility
