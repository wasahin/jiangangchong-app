# Document Writing Guide

## Description

Primary skill for document and content writing. Governs intent interpretation, genre and format selection, writing-style calibration, content structuring, and visual generation guidance.

## When to Use

Trigger on:
- PRD (Product Requirements Document)
- MVP spec, feature spec
- Technical proposal
- Research report, competitive analysis
- Manuals, user guides
- Any structured written deliverable

## Genre & Format Selection

| Document Type | Format | Key Sections |
|---|---|---|
| PRD | Structured doc | Problem, Goals, Requirements, Scope, Timeline |
| Tech Proposal | Decision doc | Context, Options, Trade-offs, Recommendation |
| Research Report | Analytical | Executive Summary, Findings, Methodology, Conclusion |
| Manual/Guide | Instructional | Overview, Steps, Examples, Troubleshooting |

## Writing Style

- **Concise**: Be direct and avoid fluff
- **Structured**: Use clear headings and logical flow
- **Actionable**: Include concrete next steps or recommendations
- **Evidence-based**: Support claims with data or citations

## Content Structuring

1. Start with a clear executive summary (1-2 sentences)
2. Use tables for comparisons
3. Use bullet lists for key points
4. Include visual aids (diagrams, charts) where helpful
