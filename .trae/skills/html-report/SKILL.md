# HTML Report

## Description

Create any self-contained HTML deliverable except slides — research reports, whitepapers, PRDs, dashboards, portfolios, resumes, email templates, data visualizations, and more.

## When to Use

Trigger this skill when the user wants to produce a polished, visually designed HTML page or multi-page site for any purpose. The final deliverable is a self-contained directory with zero external dependencies.

## Instructions

1. **Self-contained output**: Generate a complete HTML file with all CSS and JS inline. No external CDN dependencies.
2. **Visual design**: Use a polished, professional design appropriate to the content type.
3. **Responsive**: Ensure the output works across desktop, tablet, and mobile.
4. **Font handling**: For Chinese content, set font to `Noto Sans CJK SC` or `WenQuanYi Micro Hei`.

## Output Structure

- `<name>.html` — Main HTML file
- `assets/` — Supporting images and resources
- `_shared/` — Shared CSS/JS modules

## Examples

### Research Report
Generate a professional research report with cover page, table of contents, executive summary, and detailed sections with charts and data.

### Dashboard
Create an interactive data dashboard with real-time indicators, charts, and summary statistics.

### Portfolio
Build a personal portfolio page with project showcase, skills section, and contact form.
