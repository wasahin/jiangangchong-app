# HTML Deck

## Description

Create stunning, animation-rich HTML slide decks from scratch. Outputs a self-contained HTML file that opens in any browser, visually impressive and pixel-perfect across all platforms.

## When to Use

Trigger this skill when the user wants to build a brand-new presentation from a topic, outline, document, or Markdown. Covers presentations, slides, keynote, or any multi-slide document created from zero.

## Instructions

1. **Self-contained output**: Single HTML file with all CSS, JS, and assets inline.
2. **Slide navigation**: Include keyboard navigation (arrow keys, space) and click/tap navigation.
3. **Animations**: Use CSS animations for slide transitions and content reveals.
4. **Visual polish**: Each slide should be visually distinct and professional.
5. **Font handling**: For Chinese content, use `Noto Sans CJK SC` or `WenQuanYi Micro Hei`.

## Slide Structure

- Title slide with presentation name and subtitle
- Content slides with clear headings and body
- Section divider slides for topic transitions
- Summary/closing slide

## Features

- **Responsive**: Works on any screen size
- **Print-friendly**: CSS print styles for PDF export
- **Presenter mode**: Speaker notes support
- **Progress indicator**: Show current slide position
