# Research Guide

## Description

Primary research and analysis scenario skill. Provides source-hierarchy rules, cross-validation methodology, search paradigms, and sub-scenario routing for research tasks.

## When to Use

Activate when the user's intent involves:
- Searching, looking up facts, researching, investigating
- Analyzing, comparing products or technologies
- Conducting competitive analysis
- Writing a research report
- Producing any evidence-based deliverable

## Source Hierarchy

1. **Primary sources**: Official documentation, RFCs, vendor specifications
2. **Secondary sources**: Reputable tech publications, peer-reviewed articles
3. **Tertiary sources**: Community discussions, blog posts (use with caution)

## Cross-Validation

- Always verify facts from at least 2 independent sources
- Prefer recent information for time-sensitive topics
- Check official docs before citing community sources

## Search Strategy

1. Start with precise, targeted queries
2. Evaluate search snippets before fetching full pages
3. Fetch authoritative sources for detailed information
4. Cross-reference findings before drawing conclusions

## Deliverable Types

- **Research Report**: Comprehensive analysis with findings, methodology, and conclusions
- **Competitive Analysis**: Side-by-side comparison of products or technologies
- **Technical Deep-Dive**: Detailed exploration of a specific technology or concept
